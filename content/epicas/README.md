# Marco General de Épicas – Sistema PIGCCT

## 1. Contexto General

El sistema PIGCCT se concibe como una plataforma integral para la gestión, registro, validación, seguimiento y visualización de acciones, medidas e indicadores asociados a la planeación y gestión territorial.  
Las épicas aquí definidas estructuran el alcance funcional del sistema, organizando los requerimientos en grandes bloques coherentes que cubren desde la administración de catálogos maestros, la operación de formularios, los flujos de validación y control, hasta la visualización geográfica, los reportes estratégicos y la experiencia de usuario.

Cada épica representa un conjunto de capacidades funcionales relacionadas, que pueden ser desarrolladas, priorizadas y gestionadas de manera independiente, pero que en conjunto garantizan la consistencia, trazabilidad y usabilidad del sistema PIGCCT.

Este documento actúa como índice maestro de las épicas, facilitando la navegación, el entendimiento global del sistema y el acceso a la documentación detallada de cada una.
