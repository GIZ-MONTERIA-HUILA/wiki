# HU-PIGCCT-SYM-215
## Épica: Estadísticas territoriales del PIGCCT
### Estadísticas por municipio
---

## DESCRIPCIÓN HISTORIA DE USUARIO

> **Como:** usuario del sistema.  
> **Quiero:** visualizar estadísticas consolidadas por municipio.  
> **Para:** analizar el número de acciones, indicadores y el estado de avance del PIGCCT a nivel local.

---

## CRITERIOS DE ACEPTACIÓN

### 1. Consolidación de información municipal
1.1 El sistema consolida la información del PIGCCT por municipio.

1.2 Las estadísticas incluyen el número total de acciones por municipio.

1.3 Se consolida el número de indicadores asociados a las acciones municipales.

1.4 El sistema agrupa los datos según el estado de avance de las acciones.

### 2. Visualización de estadísticas
2.1 Las estadísticas se presentan mediante gráficos, tablas o indicadores numéricos.

2.2 La visualización es clara, comprensible y consistente con el diseño del sistema.

2.3 El usuario puede identificar fácilmente diferencias entre municipios.

2.4 Los elementos visuales permiten comparar estados de avance entre municipios.

### 3. Estados de avance
3.1 El sistema clasifica las acciones por estado de avance (por ejemplo: planeada, en ejecución, finalizada).

3.2 Cada estado se representa visualmente de forma diferenciada.

3.3 Las estadísticas reflejan únicamente información válida según el rol del usuario.

### 4. Interacción y navegación
4.1 El usuario puede seleccionar un municipio para ver sus estadísticas específicas.

4.2 Al seleccionar un municipio, las estadísticas se actualizan automáticamente.

4.3 El sistema permite regresar fácilmente a la vista general consolidada.

### 5. Integración con visor geográfico
5.1 Las estadísticas pueden vincularse al visor geográfico del sistema.

5.2 Al seleccionar un municipio en el visor, se actualizan las estadísticas consolidadas.

5.3 El sistema mantiene coherencia entre la información estadística y la espacial.

### 6. Actualización y consistencia de datos
6.1 Las estadísticas se generan a partir de la información más reciente disponible.

6.2 El sistema asegura la consistencia entre datos estadísticos y administrativos.

6.3 Los cambios en acciones o indicadores se reflejan automáticamente en las estadísticas.

### 7. Rendimiento y acceso
7.1 La visualización de estadísticas se carga en tiempos adecuados.

7.2 El sistema controla el acceso a la información según el rol del usuario.

7.3 La información sensible se presenta de acuerdo con las políticas de seguridad.

---

## Resultado esperado

El usuario puede consultar estadísticas consolidadas por municipio que reflejan de manera clara y actualizada el número de acciones, indicadores y su estado de avance, facilitando el análisis territorial y la toma de decisiones informadas sobre la implementación del PIGCCT a nivel local.
