# HU-PIGCCT-SYM-207
## Épica: Visor geográfico del PIGCCT 
### Visualizar acciones en el mapa
---

## DESCRIPCIÓN HISTORIA DE USUARIO

> **Como:** usuario autorizado del sistema.  
> **Quiero:** visualizar en el mapa las acciones del PIGCCT representadas espacialmente por municipio o cobertura departamental.  
> **Para:** identificar su distribución territorial.

---

## CRITERIOS DE ACEPTACIÓN

### 1. Representación territorial
1.1 El sistema permite visualizar las acciones del PIGCCT agregadas por municipio.

1.2 El sistema permite visualizar las acciones del PIGCCT agregadas por cobertura departamental.

1.3 El usuario puede alternar entre la visualización municipal y departamental.

### 2. Simbología y visualización
2.1 Las acciones se representan mediante simbología clara y diferenciada según su tipo o estado.

2.2 Los colores y símbolos cumplen criterios de legibilidad y accesibilidad.

2.3 La leyenda del mapa se actualiza de acuerdo con la visualización activa.

### 3. Interacción con el mapa
3.1 El usuario puede seleccionar un municipio o departamento para ver el detalle de las acciones asociadas.

3.2 Al seleccionar una entidad geográfica, el sistema resalta visualmente el área correspondiente.

3.3 El mapa responde de forma fluida a las interacciones del usuario.

### 4. Consulta de información
4.1 Al hacer clic sobre un municipio o cobertura departamental, el sistema muestra la información relevante de las acciones del PIGCCT.

4.2 La información presentada es coherente con los datos almacenados en el sistema.

4.3 El usuario puede acceder a información resumida o detallada según su perfil.

### 5. Integración con otros módulos
5.1 La visualización espacial se integra con los tableros e indicadores del PIGCCT.

5.2 Los filtros aplicados en el sistema se reflejan automáticamente en el mapa.

5.3 Los cambios de visualización no afectan el funcionamiento de otros módulos.

### 6. Rendimiento y consistencia
6.1 La carga de la información espacial se realiza en tiempos aceptables.

6.2 El sistema maneja correctamente grandes volúmenes de datos territoriales.

6.3 La visualización es consistente entre sesiones y usuarios con el mismo rol.

---

## Resultado esperado

El usuario puede identificar claramente la distribución territorial de las acciones del PIGCCT, visualizándolas en el mapa por municipio o a nivel departamental, lo que facilita el análisis espacial, el seguimiento y la toma de decisiones.
