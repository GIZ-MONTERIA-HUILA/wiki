# HU-PIGCCT-SYM-218
## Épica: Estadísticas territoriales del PIGCCT
### Estadísticas por eje y medida
---

## DESCRIPCIÓN HISTORIA DE USUARIO

> **Como:** usuario del sistema.  
> **Quiero:** consultar estadísticas territoriales agrupadas por eje y medida.  
> **Para:** analizar el enfoque estratégico de las acciones del PIGCCT en el territorio.

---

## CRITERIOS DE ACEPTACIÓN

### 1. Agrupación por eje
1.1 El sistema agrupa las acciones e indicadores por eje estratégico del PIGCCT.

1.2 La agrupación por eje se puede visualizar a nivel municipal y departamental.

1.3 Cada eje muestra el total de acciones, indicadores asociados y su estado.

1.4 Los datos agrupados reflejan únicamente información válida según el rol del usuario.

### 2. Agrupación por medida
2.1 El sistema agrupa las acciones e indicadores por medida dentro de cada eje.

2.2 Cada medida muestra el número de acciones asociadas y su nivel de avance.

2.3 La relación eje–medida se mantiene clara y jerárquica en la visualización.

2.4 El sistema permite identificar fácilmente las medidas con mayor o menor presencia territorial.

### 3. Visualización estadística
3.1 Las estadísticas se presentan mediante gráficos, tablas o paneles resumen.

3.2 El usuario puede alternar entre visualizaciones por eje o por medida.

3.3 La información se presenta de forma clara, ordenada y comprensible.

3.4 La visualización facilita la comparación entre ejes y medidas.

### 4. Enfoque territorial
4.1 El sistema permite seleccionar un municipio o departamento para consultar sus estadísticas.

4.2 Las estadísticas reflejan la distribución territorial de los ejes y medidas.

4.3 El usuario puede identificar qué ejes o medidas predominan en cada territorio.

4.4 La información apoya el análisis estratégico territorial del PIGCCT.

### 5. Integración con el visor geográfico
5.1 Las estadísticas por eje y medida se integran con el visor geográfico.

5.2 El mapa puede reflejar la concentración de acciones por eje o medida mediante simbología temática.

5.3 Al interactuar con el mapa, se actualizan las estadísticas mostradas.

5.4 Existe coherencia entre la información estadística y la representación espacial.

### 6. Filtros y segmentación
6.1 El usuario puede filtrar las estadísticas por eje, medida, municipio o departamento.

6.2 El sistema permite combinar filtros para análisis más específicos.

6.3 Los filtros se aplican dinámicamente sin recargar la página.

6.4 Los resultados se actualizan automáticamente al modificar los filtros.

### 7. Control de acceso
7.1 El sistema muestra estadísticas según los permisos del usuario.

7.2 Los usuarios solo visualizan información autorizada según su rol.

7.3 La información sensible se protege conforme a las políticas de seguridad.

### 8. Rendimiento y actualización
8.1 Las estadísticas se generan a partir de datos actualizados y validados.

8.2 El tiempo de respuesta del sistema es adecuado para la consulta.

8.3 La carga de estadísticas no afecta el rendimiento general del sistema.

---

## Resultado esperado

El usuario puede consultar estadísticas territoriales agrupadas por eje y medida, presentadas de forma clara y coherente, que permiten analizar el enfoque estratégico de las acciones del PIGCCT en los diferentes territorios y apoyar la toma de decisiones informadas.
