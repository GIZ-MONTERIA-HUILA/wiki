# HU-PIGCCT-SYM-072
## Épica: Gestión de acciones territoriales y seguimiento de indicadores del PIGCCT
### Registrar valores proyectados

---

## DESCRIPCIÓN HISTORIA DE USUARIO

> **Como:** usuario del sistema.                            
> **Quiero:** registrar el valor proyectado de cada seguimiento.                      
> **Para:** planificar el cumplimiento del indicador.

## CRITERIOS DE ACEPTACIÓN

### 1. Registro del valor proyectado

1.1 El sistema debe permitir registrar el valor_proyectado para cada seguimiento asociado a un indicador.                     
1.2 El valor_proyectado debe ser almacenado como un dato válido.

### 2. Habilitación de campos

2.1 El sistema debe habilitar únicamente el campo valor_proyectado para su diligenciamiento.             
2.2 Los demás campos del seguimiento deben permanecer deshabilitados.

### 3. Consistencia de la información

3.1 El valor_proyectado debe quedar asociado al registro de seguimiento correspondiente.              
3.2 La información debe estar disponible para los módulos de seguimiento y análisis.

### 4. Usabilidad y experiencia de usuario

4.1 La interfaz debe ser clara, intuitiva y consistente con el diseño general del sistema.
4.2 El sistema debe permitir cancelar la operación sin guardar cambios.

### Resultado esperado

El sistema permite registrar únicamente los valores proyectados de los seguimientos, garantizando una planificación clara y consistente del cumplimiento de los indicadores del PIGCCT.
