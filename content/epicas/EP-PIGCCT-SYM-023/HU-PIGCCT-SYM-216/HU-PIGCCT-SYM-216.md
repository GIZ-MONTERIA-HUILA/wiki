# HU-PIGCCT-SYM-216
## Épica: Estadísticas territoriales del PIGCCT
### Estadísticas por departamento
---

## DESCRIPCIÓN HISTORIA DE USUARIO

> **Como:** usuario del sistema.  
> **Quiero:** visualizar estadísticas consolidadas por departamento.  
> **Para:** evaluar el desempeño territorial del PIGCCT a nivel regional.

---

## CRITERIOS DE ACEPTACIÓN

### 1. Consolidación de información departamental
1.1 El sistema consolida la información del PIGCCT a nivel departamental.

1.2 Las estadísticas incluyen el número total de acciones por departamento.

1.3 El sistema consolida los indicadores asociados a las acciones departamentales.

1.4 Los datos se agrupan por estado de avance de las acciones.

### 2. Visualización de estadísticas

2.1 Las estadísticas se presentan mediante gráficos, tablas o indicadores clave.

2.2 La información se muestra de forma clara, intuitiva y consistente con el diseño del sistema.

2.3 El usuario puede comparar fácilmente el desempeño entre departamentos.

2.4 La visualización permite identificar departamentos con mayor o menor avance.

### 3. Estados de avance
3.1 El sistema clasifica las acciones por estado de avance (por ejemplo: planeada, en ejecución, finalizada).

3.2 Cada estado se representa mediante simbología o codificación visual diferenciada.

3.3 Las estadísticas muestran únicamente información autorizada según el rol del usuario.

### 4. Interacción y navegación
4.1 El usuario puede seleccionar un departamento específico para consultar su detalle estadístico.

4.2 Al seleccionar un departamento, las estadísticas se actualizan automáticamente.

4.3 El sistema permite regresar fácilmente a la vista general nacional o consolidada.

### 5. Integración con visor geográfico
5.1 Las estadísticas departamentales se integran con el visor geográfico del sistema.

5.2 Al seleccionar un departamento en el mapa, se reflejan sus estadísticas correspondientes.

5.3 El sistema mantiene coherencia entre la información espacial y estadística.

### 6. Actualización y consistencia de datos
6.1 Las estadísticas se generan a partir de la información más reciente del sistema.

6.2 Cualquier actualización en acciones o indicadores se refleja automáticamente en los consolidados departamentales.

6.3 El sistema garantiza la consistencia entre la información estadística y administrativa.

### 7. Rendimiento y control de acceso
7.1 La carga de estadísticas departamentales se realiza en tiempos adecuados.

7.2 El acceso a la información se controla según el perfil del usuario.

7.3 Los datos sensibles se visualizan conforme a las políticas de seguridad establecidas.

---

## Resultado esperado

El usuario puede consultar estadísticas consolidadas por departamento que reflejan claramente el número de acciones, indicadores y su estado de avance, permitiendo evaluar el desempeño territorial del PIGCCT a nivel regional y apoyar la toma de decisiones estratégicas.
